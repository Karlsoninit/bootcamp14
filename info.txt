---------------------------- отображение главной страницы ----------

базовывй url для запроса --- https://dash-ads.goit.co.ua/api/v1

Методы для всплывающих оповещений, должны появлятся при добавлении, удалении --- axios.get(https://sciactive.com/pnotify/ )

Полуение всех обьявлений в нашей базе, для отрисовки на стартовой страницы axios.get('https://dash-ads.goit.co.ua/api/v1/ads/all')
Получаем массыв с возможностью добавления дополнительных параметров 

        "totalDocs": 50,
        "limit": 10, --- максимальное количество обьявлений при первом запросе 
        "hasPrevPage": false, --- наличие возможности перейти на предведущую стрпницу (нужно для отображения кнопки "вперед" или "назад")
        "hasNextPage": true,  --- наличие возможности перейти на предведущую стрпницу (нужно для отображения кнопки "вперед" или "назад")
        "page": 1, --- страница 
        "totalPages": 5, --- всего страниц 
        "pagingCounter": 1, --- сколько всего страниц отображается 
        "prevPage": null, --- наличие предведущей страницы 
        "nextPage": 2, --- какая следующая страница
        
Получение обьявления по  id --- axios.get(https://dash-ads.goit.co.ua/api/v1/ads/{ // указываем id обьявления 5d8cdf235c35f91a27d75b8f }) , у каждого обьявления есть id который можно использвать для отобаржении расшыренной информации дпри клике на карточку с товаром 

Выбор лимита выдачи обявлений --- axios.get(https://dash-ads.goit.co.ua/api/v1/ads/all?limit={ лимит }&page={ ноемр старницы  })

Получение обьявлений по категориям axios.get(https://dash-ads.goit.co.ua/ads/all?category=${ номер категории }&page=${ ноемр старницы })


---------------------------- авторизация  ----------

регистрация юзера --- axios.post(https://dash-ads.goit.co.ua/auth/register`, {
      email: email,
      password: password,
      name: name,
    })

при регистрации пользывателя в ответ получаем обьект в котором будет вся служебная информация про юзера ! все его обьявления и его избранные, а так же token



вход для зарегистрированных юзеров --- axios.post(`https://dash-ads.goit.co.ua/auth/login`, {
    email: "test9@gmail.com",
    password: "qwerty"
})

в ответ получаем такой же обьект как и при регистрациии + обязательно указать emaol и login рпи запросе иначе будет ошибка, также оштбка будет при не правильном вводе 

выход пользователя axios.post(`https://dash-ads.goit.co.ua/auth/logout`, 
    { email: email,
      password: password,
    }, {
      headers: {
      Authorization: token,
        })

обязательно указать оьбект с email и password который вводили для входа в приложение и передать token, произойдет розлогированик
и старый token больше не будет подходить для входа


---------------------------- добавление обьявлений  ----------


крточка для добавления товара ---  axios.post(`https://dash-ads.goit.co.ua/ads`, {
    images : '',
    title: '',
    category: '',
    price: ',
    pho: '',
    description : ''
    }, {
        headers: {
          Authorization: this.userToken,
        },
      })  одна для всех с обязательными полями для заоплнения, при добавлении обьявления оно автоматически будел лежать вверху общего поиска


удаление обьявления --- axios.delete(`https://dash-ads.goit.co.ua/ds/${adId}`, {
        headers: {
          Authorization: token,
        },
      });
передаем token юзера и по нему удаляем
